# Measurement Template System User Guide

This guide provides instructions on how to use the measurement template system in the Sartor Order Management app.

## Creating a New Measurement from a Template

1.  Navigate to the "Measurements" screen and tap the "Add" button.
2.  Tap the "Create from Template" button.
3.  Select a template from the grid.
4.  The measurement form will be pre-filled with the template's measurements.
5.  Make any necessary adjustments to the measurements.
6.  Enter a profile name and any notes.
7.  Tap the "Save" button to save the new measurement.

## Resetting a Measurement to its Template

1.  Navigate to the "Measurements" screen and select a measurement that was created from a template.
2.  Tap the "Edit" button.
3.  Tap the "Reset to Template" button.
4.  The measurements will be reset to the original template values.
5.  Tap the "Save" button to save the changes.

## Admin Procedures

### Accessing the Template Management Screen

1.  From the main menu, navigate to the "Admin" section.
2.  Select "Measurement Templates" to open the template management screen.

### Creating a New Template

1.  Tap the "Add" icon in the top right corner of the screen.
2.  Fill in the template details:
    *   **Name:** A descriptive name for the template (e.g., "Standard Shirt", "Slim Fit Trousers").
    *   **Description:** An optional description for the template.
    *   **Measurements:** Enter the values for each measurement field.
3.  Tap the "Save" button to create the new template.

### Editing an Existing Template

1.  Find the template you wish to edit in the list.
2.  Tap the "Edit" icon next to the template name.
3.  Modify the template details as needed.
4.  Tap the "Save" button to apply the changes.

### Duplicating a Template

1.  Find the template you wish to duplicate in the list.
2.  Tap the "Copy" icon next to the template name.
3.  A new template will be created with the same details as the original.

### Deleting a Template

1.  Find the template you wish to delete in the list.
2.  Tap the "Delete" icon next to the template name.
3.  Confirm the deletion when prompted.

### Importing Templates

1.  Tap the "Upload" icon in the top right corner of the screen.
2.  Select a CSV file containing the template data.
3.  The templates from the CSV file will be added to the system.

### Exporting Templates

1.  Tap the "Download" icon in the top right corner of the screen.
2.  Choose a location to save the `measurement_templates.csv` file.
3.  All existing templates will be exported to the selected CSV file.

### Template Analytics

1.  Tap the "Analytics" icon in the top right corner of the screen.
2.  The analytics screen displays two reports:
    *   **Template Usage:** Shows how many times each template has been used to create a new measurement.
    *   **Template Conversion Rates:** Shows the percentage of times a template is used without modification versus with modification.
